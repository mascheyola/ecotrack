\n# EcoTrack Blueprint\n\n## Overview\n\nEcoTrack is a mobile application designed to empower users to track their recycling habits, view their environmental impact, and discover recycling best practices. The app provides a user-friendly interface for registering recycled materials, visualizing personal statistics, and finding local recycling points.\n\n## Implemented Features\n\n### Authentication\n\n*   **User Registration:** New users can sign up with their name, email, and password.\n*   **User Login:** Existing users can sign in with their email and password.\n*   **Session Management:** The app securely manages user sessions, keeping them logged in until they explicitly log out.\n\n### Home Screen\n\n*   **Personalized Greeting:** The screen welcomes the user with a "Hola, \[user name]" message.\n*   **2x2 Grid Layout:** A visually appealing grid of four large, square buttons provides access to the main features.\n*   **Button Styling:** Buttons use the app\'s primary color scheme (`deepPurple`) and feature prominent icons and bold text for clarity.\n*   **Logout:** A logout button is conveniently located in the `AppBar`.\n\n### Core Features\n\n*   **Recycling Registration:** A dedicated screen allows users to log the types and quantities of materials they recycle.\n*   **Statistics:** Users can view their recycling history and performance through charts and data visualizations.\n*   **Recycling Tips:** A screen provides helpful tips and best practices for recycling.\n*   **Recycling Points:** Users can find nearby recycling centers and drop-off locations.\n\n### Theming\n\n*   **Material Design 3:** The app uses the latest Material Design guidelines for a modern look and feel.\n*   **Light & Dark Modes:** A theme provider allows users to switch between light and dark themes, or follow the system setting.\n*   **Custom Fonts:** The `google_fonts` package is used for a unique and readable typography.\n\n## Current Plan\n\n*   **Implement functionality for the four main feature screens:** Recycling Registration, Statistics, Recycling Tips, and Recycling Points.\n*   **Connect the app to a Firebase backend** to store and retrieve user data, recycling logs, and other relevant information.\n*   **Flesh out the UI/UX** for each screen to ensure a polished and intuitive user experience.\n
