# EcoTrack Blueprint

## Overview

EcoTrack is a mobile application designed to help users track their recycling habits and discover recycling locations. The app will feature a personalized dashboard, a map to find recycling centers, and a sharing feature to promote recycling among friends and family.

## Style and Design

- **Theme:** The app will use a green and white color scheme to evoke a sense of nature and cleanliness.
- **Typography:** The app will use the Google Fonts package to implement the Oswald and Roboto fonts for a clean and modern look.

## Features

### Implemented Features

- **Dashboard:** A personalized dashboard that displays the user's recycling statistics.
- **Recycling Map:** A map that displays nearby recycling centers.
- **Sharing:** A feature that allows users to share their recycling achievements.

### Current Plan

- **App Name:** Update the app name to "EcoTrack" in the `AndroidManifest.xml` file.
- **App Icon:** Add a recycling icon to the app.
